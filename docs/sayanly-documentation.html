Product Overview

Sayanly is an AI-powered weekly social media content factory. A business enters their company profile once — and Sayanly researches trending topics, writes platform-optimized posts, and generates ready-to-post graphics for a full week in minutes.

The Problem

Businesses and agencies spend 5–10 hours per week creating social media content. Writing posts, researching what topics are trending, designing graphics, adapting content for each platform — it's repetitive, time-consuming, and often inconsistent in quality.

The Solution

Sayanly automates the entire weekly content workflow using AI. One profile setup → one click → a full week of posts and graphics, ready to schedule.

Target Users

Small Business Owners

Managing their own social media without a team or agency budget.

Marketing Managers

At SMEs who need consistent content across platforms every week.

Social Media Agencies

Managing multiple client accounts — Sayanly scales content production.

Consultants & Personal Brands

Solo professionals who need to maintain a consistent online presence.

Platforms Supported

LinkedIn Facebook Instagram

Each platform gets its own version of every post — tone, length, hashtag style, and formatting are all adapted automatically.

Core Features

1 Company Profile

Set up once. Used for all future weeks. Stores company name, niche, target audience, tone of voice, brand colors, platforms, past posts, and competitor posts.

2 Topic Research Engine

4-method hybrid: Claude's industry knowledge + competitor gap analysis + Tavily real-time web search + user's own post history. Suggests 7 topics per week with reasoning.

3 Weekly Post Generator

For each approved topic → generates 3 platform-specific versions (LinkedIn, Facebook, Instagram). Each includes hook, body, CTA, 15 hashtags, and a first comment.

4 Graphic Generator

Claude writes a detailed image prompt matched to the post and brand colors. Pollinations AI generates the actual graphic (free, no API key needed). User downloads ready-to-post images.

5 Poster Cloner (Phase 2)

User uploads any graphic they like. Claude Vision analyzes layout, style, composition, and colors. Recreates the same design with user's brand colors and company details.

6 Week History & Export

Every generated week is saved. User can revisit past weeks, export all posts as PDF, or bulk-copy posts per platform.

4

Topic Research Engine

The topic engine combines 4 independent research methods and merges them into 7 final topic suggestions with reasoning.

Method 1 — Claude Knowledge

Claude's training data covers industry trends, pain points, and content formats for virtually any niche. Always active as the base layer.

Method 2 — Competitor Analysis

User pastes competitor posts in their profile. Claude identifies what topics they cover, finds gaps, and suggests angles competitors have missed.

Method 3 — Web Search (Tavily)

Real-time search for trending topics in the company's niche this week. Pulls current conversations, news, and pain points from LinkedIn and Google.

Method 4 — Past Post Analysis

User pastes their own previous posts. Claude avoids repetition, builds on what worked, and ensures content variety week to week.

Topic Output Format

Field Description Example
Title Topic headline "Why most startups fail at hiring"
Reasoning Why this will perform this week "Trending after recent layoff news"
Best Platform Which platform suits this topic LinkedIn
Format Post format suggestion List / Opinion / Story / Stat / Question

Post Generator

For each approved topic, the system generates 3 platform-specific post versions in a single Claude API call.

Platform Differences

Platform Tone Length Style
LinkedIn Professional, insight-driven Longer form Ends with thought-provoking question
Facebook Conversational, community Medium Story-driven, relatable
Instagram Punchy, visual-first Short caption Emoji-friendly, strong CTA

Post Structure (per platform)

Hook

First line — designed to stop scrolling. Question, stat, or bold statement.

Body

2–4 sentences or bullet points. The value delivery.

CTA

Clear call to action. Comment, share, follow, or visit.

Hashtags + First Comment

15 platform hashtags + 10 niche hashtags in first comment for reach boost.

Output per week: 7 topics × 3 platforms = 21 ready-to-post pieces of content, all generated in under 30 seconds.

Data Model

type CompanyProfile = {
  id: string
  name: string                  // Company name
  description: string          // What the company does
  target_audience: string      // Who they sell to
  tone: 'professional' | 'casual' | 'inspirational' | 'educational' | 'bold'
  brand_colors: { primary: string; secondary: string }
  platforms: ('linkedin' | 'facebook' | 'instagram')[]
  past_posts?: string          // Optional — improves topic research
  competitor_posts?: string   // Optional — finds content gaps
  created_at: string
}

type Topic = {
  id: string
  title: string
  reasoning: string            // Why this will perform
  best_platform: string
  format: string               // List / Opinion / Story / Stat
  approved: boolean
}

type Post = {
  topic_id: string
  linkedin:  PlatformPost
  facebook:  PlatformPost
  instagram: PlatformPost
}

type PlatformPost = {
  hook: string
  body: string
  cta: string
  hashtags: string[]            // 15 tags
  first_comment: string        // 10 niche tags
}

type Week = {
  id: string
  profile_id: string
  week_label: string           // e.g. "Week of May 12, 2026"
  created_at: string
  topics: Topic[]
  posts: Post[]
  graphics: Graphic[]
  status: 'draft' | 'complete'
}
